# ADR-002 — State management: Zustand v4

Status: Accepted (2026-07-17)

Zustand stores are plain serialisable objects, which makes JSON export/import and the determinism guarantee straightforward — the store snapshot *is* the session payload. Redux Toolkit adds ceremony without adding needed guarantees; Jotai's atom granularity fits derived UI state but complicates whole-session serialisation. Zustand's selector subscriptions keep large holdings tables performant. Middleware slot cleanly hosts the audit-event emitter.
