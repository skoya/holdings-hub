# ADR-003 — Routing on GitHub Pages: HashRouter

Status: Accepted (2026-07-17)

GitHub Pages serves static files with no rewrite rules, so `BrowserRouter` deep links 404 unless a `404.html` copy-hack redirects with scripted URL rewriting — fragile, bad for shared links, and it flashes the 404 page. `HashRouter` makes every deep link a real, refresh-safe URL with zero server cooperation. Cosmetic URLs are not a requirement for a simulation prototype, so the hash is an acceptable trade.
