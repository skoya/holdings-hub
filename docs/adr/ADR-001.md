# ADR-001 — Framework: React 18 + Vite 5 + TypeScript strict

Status: Accepted (2026-07-17)

React is required in practice by the chosen diagram (`@xyflow/react`) and accessibility (`@radix-ui`) libraries, which have no first-class Vue/Svelte equivalents of the same maturity. Vite produces clean static bundles ideal for GitHub Pages and shares config with Vitest. Vanilla JS was rejected because the app's state surface (sessions, entitlements, deterministic engine) demands a typed component model; Svelte/Vue were rejected on ecosystem fit, not quality. Strict TypeScript is non-negotiable given runtime JSON import boundaries.
