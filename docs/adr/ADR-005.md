# ADR-005 — Diagram libraries: `@xyflow/react` v12 + D3 v7

Status: Accepted (2026-07-17)

Process/state-machine diagrams need React-integrated nodes with pan/zoom and JSON-definable graphs — xyflow v12 is the maintained successor to React Flow and does exactly this. Entity relationship graphs need physics layout and scale, which is D3's home ground; a canvas/SVG hybrid keeps large graphs fast while preserving accessible interactive labels. One library could not serve both well: xyflow has no force layout of D3's quality; raw D3 process diagrams would reimplement xyflow. Mermaid/Cytoscape were rejected as less React-native and harder to animate against the engine's state machine.
