# ADR-007 — Persistence: `localforage`

Status: Accepted (2026-07-17)

Session libraries with audit logs and pinned layouts will exceed localStorage's ~5 MB comfort zone; IndexedDB removes that ceiling and localforage falls back to localStorage automatically where IndexedDB is blocked (private browsing). Plain localStorage was rejected on quota; sessionStorage was rejected because the Simulation Library must survive browser restarts. The async API also keeps large session writes off the main thread's critical path.
